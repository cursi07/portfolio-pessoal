Projeto: Portfólio Pessoal


Tipo: Portfólio Web


Status: Concluído


Tecnologias: HTML, CSS e JavaScript


Descrição:

Desenvolvimento de um portfólio pessoal responsivo para apresentação profissional, reunindo informações sobre minha trajetória, experiências, projetos e formas de contato. O projeto foi desenvolvido com foco em organização, design moderno, responsividade e boa experiência do usuário, servindo como vitrine para demonstrar minhas habilidades e evolução no desenvolvimento web.


Principais funcionalidades:

  • Layout moderno e responsivo.
  • Apresentação da trajetória profissional.
  • Exibição de experiências e projetos.
  • Download do currículo em PDF.
  • Links para LinkedIn, GitHub e WhatsApp.
  • Animações e efeitos visuais com HTML e CSS.
  • Navegação entre páginas do portfólio.
